Smart Campus Security Monitoring System

Designed a security monitoring system that analyzes login activity, system logs, and alerts to identify suspicious behavior across campus devices. The project focused on detecting anomalies such as unusual login locations, repeated failed attempts, and unauthorized file changes. It provided summarized alerts and basic risk scoring to help administrators respond faster to potential security incidents.

 

Internship & Job Application Tracker Web App

Built a web-based application to help students track job and internship applications in one place. Users could log company details, application status, deadlines, and interview dates. The project aimed to reduce missed deadlines and improve organization during job searches, especially for students applying to multiple roles at once.
