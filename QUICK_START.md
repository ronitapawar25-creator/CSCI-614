### Q: Changes not showing?

A: Check DevTools device mode (Ctrl+Shift+M)

### Q: Animation too fast/slow?
A: Edit `--transition-base: 300ms` in `:root`

### Q: Colors not changing?
A: Make sure variable name matches exactly in `:root`
